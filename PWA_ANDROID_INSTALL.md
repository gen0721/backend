# Установка Minions Market как Android приложение (PWA)

## Что такое PWA?
Progressive Web App (PWA) - это веб-приложение, которое может устанавливаться на устройство как нативное приложение.

## Преимущества PWA версии:
- ✅ Не требует Google Play
- ✅ Автоматические обновления
- ✅ Меньше места занимает
- ✅ Работает оффлайн
- ✅ Быстрый запуск
- ✅ Полноэкранный режим

## Как установить на Android:

### Способ 1: Через браузер Chrome
1. Откройте сайт Minions Market в Chrome браузере
2. Нажмите на меню (три точки) в правом верхнем углу
3. Выберите "Добавить на главный экран" или "Установить приложение"
4. Подтвердите установку
5. Приложение появится на главном экране

### Способ 2: Через уведомление
1. Откройте сайт Minions Market
2. Дождитесь появления баннера "Установить приложение"
3. Нажмите "Установить"
4. Подтвердите установку

## Что нужно сделать перед установкой:

### 1. Сгенерировать иконки
Откройте файл `frontend/public/generate-icons.html` в браузере и нажмите "Сгенерировать все иконки". Переместите скачанные иконки в папку `frontend/public/`.

### 2. Собрать проект
```bash
# В корневой папке проекта
npm run build

# Или отдельно:
cd frontend && npm install && npm run build
```

### 3. Запустить сервер
```bash
cd backend && npm install && npm start
```

### 4. Открыть сайт в браузере
Перейдите по адресу `http://localhost:5000` в Chrome на Android устройстве.

## Проверка установки:
После установки приложение будет:
- Открываться в полноэкранном режиме
- Иметь собственную иконку на главном экране
- Работать без интернет-соединения (кэшированные страницы)
- Отправлять push-уведомления (если настроено)

## Удаление PWA:
Удалите как обычное приложение - долгое нажатие на иконку → "Удалить".

## Технические требования:
- Android 5.0+ 
- Chrome 70+ (или другой Chromium браузер)
- HTTPS соединение (для production)

## Для разработки:
- Используйте `npm run dev` для локальной разработки
- PWA функции работают только через HTTPS или localhost
- Проверяйте в Chrome DevTools → Application → Manifest
